# Aula 01 - Fundamentos do SQL

**O que são dados?**

- Uma informação sobre algo que podemos chegar a uma conclusão.
- Dados são extremamente importante para as empresas.


**O que é um banco de dados?**

- O conjunto de dados gera uma tabela. Um conjunto de tabelas gera um Banco de Dados. 


**BD Relacional**

- Conjunto de tabelas com dados relacionados.


**Sistema de Banco de Dados**

- Programa que permite a manipulação de banco de dados.
- Exemplos: MySQL, Oracle, SQL Server, PostgreSQL


**O que é o SQL?**

- SQL = Structured Query Language 
- É uma linguagem de programação usada para manipular dados.


**O que é uma query?**

- Consulta ao banco de dados sobre algo.


**Por que aprender SQL?**

- Futuro, pois os dados estão sendo cada vez mais valorizado.
- Segurança de dados: é um sistema de proteção de dados robusto.
- Linguagem atempora: funciona mesmo com o avanço do tempo.


**Profissionais em Banco de Dados**

- Administrador de Banco de Dados (AD);
- Desenvolvedor de Banco de Dados (DB);
- Analista Business Intelligence; 
- Análise de Dados;
- Cientista de Dados.
