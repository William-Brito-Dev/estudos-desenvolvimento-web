Aula 04 - Primeiros comandos em Python3 

- print é usado para saída de dados (mostrar uma informação na tela).
- Todos os comandos em Python3 são funções e toda função deve estar em parenteses ( ).
- Toda string deve estar entre aspas simples ' '. 


Mensagem: usados para exibir mensagens na tela.
- Ex: print ('isto é uma mensagem!') -> isto é uma mensagem!
- Ex: print ('7'+'4') -> 74

Números: usados para operações aritméticas.
- Ex: print (7+5) -> 11

- Usamos a virgula "," para juntar strings e números.


## Variáveis 

- Sempre deve ser declarada com letras minusculas (kamel case)
- Para o python3 toda variável é um objeto. Um objeto é mais do que uma variável.
- Toda variável pode receber um valor.

Atribuição de dados a uma variável usa-se o "=". 

- Ex: nome = 'william'
- Ex: peso = 73
- Ex: idade = 20


Para entrada de dados usamos o input.

- Ex: nome = input ('qual é o seu nome?')
- Ex: peso = input ('qual é o seu peso?')
- Ex: idade = input ('qual é sua idade?')

## Aula 06 - Tipos primitivos e saídas de dados 

Existem básicamente 4 tipos de dados primitivos: 

- **str:** todo dado declarado dentro de aspas. Ex: 'isso é uma string'.
- **int:** aceita apenas valor inteiro. Ex: 7, -8, 63, 98568.
- **float:** aceita valores reais. Ex: 4.5, 0.007, -15.223
- **bool:** aceita apenas verdadeiro ou falso Ex: False or True. 

Outra maneira de utilizar o print 

- print (f'informação {}') - método f-string. 

- Ao juntar uma string em outra estamos usando a concatenação.
- Usamos o type para exibir o tipo de dado está sendo utilizado.

- usamos os métodos de teste de tipos (métodos de strings) para saber o tipo de dados que estamos lidando.

Aula 07 - Operadores aritméticos 

- No Python o limite é o tamanho memória.
- Podemos usar o 'pow'para calcular pontência. Ex: pow (3,4)
- Ao usar o pow, perdemos a ordem de procedência.
- Calcular a raiz de um número fazendo a pontência vezes meio. Ex: 81**(1/2)

Podemos usar alinhamentos:

- :> - alinha a direita.
- :< - alinha a esquerda.
- :^ - alinha ao centro. 
- := - adiciona '='.

Operador de string 

- Podemos usar a multiplicação de um valor. Ex: 'oi'*5 == 'oioioioioi'  


- Usamos variáveis se precisarmos usar o resultado depois. Senão, não há necessidade.
- Podemos formatar os valores floats usando :. valorf. Ex: {:.2f}.
- Para não quebrar a linha ao usar vários prints, adicionamos o end = ''.
- Podemos quebrar a linha ao adicionarmos o \n no código.
