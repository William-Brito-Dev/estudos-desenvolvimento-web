**Aula 04 - A evolução do PHP** 

- Início da internet em 1970
- Inicio da WWW em 1990
- Em 1995 Rasmus Lerdorf cria uma interface para saber quantos visitantes estavam acessando seu site.
- Batiza sua interface como Personal Home Page Tools - primeira versão do PHPT.
- Cria uma maneira de usuário interagirem com os devs por meio de um formulário
- Renomeia para Personal Home Page Forms Interpreter - segunda versão do PHPFI
- Para que sua ferramenta crescesse ele liberou o acesso do código-fonte para outros devs.
- PHP/FI recebe acesso a Banco de Dados e recebeu novas funcionalidades.
- 1997 mais de 60mil dominios já usavam o PHP.
- Zeev Suraski e Andi Gutmans reescrevem do zero o PHP dando a ele características de linguagens de programação.
- Reenomeiam o nome para PHP: Hypertext Processor - terceira versão.
- 2000 surge a versão 4 com correção de alguns bugs
- 2004 surge a versão 5 com 32 versões.
- 2006 surge  a premissa da versão 6 com o intuito de resover os problemas com  Unicode.
- 2013 surge a versão 5.5 do PHP
- acrônimo recursivo
