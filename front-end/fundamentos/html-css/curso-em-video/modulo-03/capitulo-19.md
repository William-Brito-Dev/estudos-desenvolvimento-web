# Aula 02 - Colocando uma imagem de fundo no seu site

- Imagens de fundo não possue sentido semântico.
- Um bom programador coda por pedaços. Faz um pouco e testa.
- Todo elemento de caixa pode receber uma imagem de fundo.
- Cuidado com tamanho do arquivo.


# Aula 03 - Imagens que se repetem no fundo do site

- Podemos colocar qualquer imagem pelo url dela.
- Tomar cuidado com imagens por link pois podem deixar de existir.
- **Background-size**: altera o tamanho da imagem de fundo.
- **Background-repeat**: repete a imagem de fundo.


# Aula 04 - Configurando a posição da imagem no fundo do site

- **Background-position**: posiciona imagem em algumas posições.
- Dividir em colunas e linhas: primeiro colunas depois linhas.


# Aula 05 - Mudando o tamanho da imagem de fundo do site

- O ideal é ter uma cor de fundo mesmo se ele tiver um fundo com imagem.
- No **background-size** pode ser usado dois valores:       
    - contain (Exibe a imagem sem corte) 
    - cover (cobre a tela inteira).


# Aula 06 - background-attachment e shorthand

- **background-attachment** cria um vínculo.
- O padrão é scroll (que rola o conteúdo).
- Fixed (fixa a imagem e o conteúdo rola).
- shorthand - background: color > image > position > repeat > (size) > attachment.
- O size é a única propriedade que não funciona na shorthand.
  

# Aula 07 - Centralização vertical de caixas

- Quando vamos centralizar algo verticalmente devemos ter um container e conteúdo.
- Margin: auto para alinhamento somente horizontal.  
- Todo posicionamento de um item boxing por padrão é relative.
- Quando temos positon absolute podemos usar left e top. 
- Podemos centralizar usando transform.
