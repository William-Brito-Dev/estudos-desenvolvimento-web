# Aula 01 - O que é Git e GitHub?

- Conhecimentos básicos em Git e Github. Necessário para colocar o projeto no ar.

**Repositório local/remoto**

- Lugar onde armazenamos nossos arquivos chamamos de repositório.
- Gerenciador de versões: guarda todas as versões do projeto. 
- Git é open source e mais usado.
- Github armazena remotamente as versões de locais do projeto criado pelo Git.

# Aula 10 - GitHub em vários PCs

**Quando estamos em outro PC**

- Ter no mínimo: Vs Code, navegador, Git e GitHub instalados.
- Para não deixar rastros:

            (Dentro do GitHub Desktop)

    - Remover o repositório;
    - Sair da conta;
    - Apagar dados do Git;
    - Excluir o arquivo da lixeira.
