## Principais comandos 

## git init - Inicía um novo repo ou inicializa um já existente.

    - --quiet - exibe apenas mensagens de erros e avisos.
    - --bare - cria um repo simples.
    - --

## git clone - Cria uma cópia local de um repo que está no Github.

## git add - Estágio de mudanças. Git começa a monitorar e registar as mudanças feitas nos arquivos do repo.

## git commit - Registra as mudanças no histórico.

## git status - mostra o status das mudanças Untracked, Modified ou Staged.

## git branch - Mostra as branches locais que estão funcionando.

## git merge - Mescla as mudanças de uma brench em outra.

## git pull - Aplica os commits feitos na brench remota ao ambiente de desenvolvimento local.

## git push - Atualiza o repositório remoto com os commit feitos localmente.
